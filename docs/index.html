Blockchain Data Engineer

We're looking for an engineer to run the blockchain nodes and extract complete data from them.

  • You extract blockchain data from nodes: sometimes APIs work, sometimes you need to trace specific transactions or even patch node code.
  • You build an ingestion pipeline from scratch that stays ahead of the incoming blockchain data.
  • You observe what breaks it and experiment with ways to keep it stable.
  • You work closely with other engineers on the team: you provide them with normalized blockchain data and they build clustering logic.

Our goal is to get data from ALL blockchains. We're talking about dozens of terabytes of data and it grows non-stop.

You own blockchain nodes infrastructure and perform low-latency data extraction. Later on, you take technical ownership of the ingestion layer and build the team around it.

Rust Backend Engineer / Graph Systems Cofounder

We're looking for a Rust Backend Engineer and performance freak to build a CPU-first RAM-intense graph analytic engine capable of operating on terabytes of data.

You build the core system where it all comes together into a single massive graph. You work with other engineers who provide you with the normalized data and clustering logic.

  • You create the in-memory structure and an API to extract subgraphs. No query language. Simple and damn fast.
  • One machine to hold it all: terabytes of RAM and tens of terabytes of SSD.
  • The data volume is huge and grows continuously.
  • Clustering and heuristics run directly on the parts of the graph affected by new data, fast enough to keep up with continuous inflow.

You own performance, data structures, algorithmic execution over the graph, and benchmarking. Early on, the focus is on building and proving the system itself. Later, this role becomes a technical lead position with freedom to hire.
